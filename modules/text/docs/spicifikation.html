Text
Modul textu. Umožňuje vložení textového pole (i s obrázky a jeho úpravu). Vhodné např. pro kontakty. Obsahuje také panel, do kterého lze napsat zkrácenou verzi obsahu.
Práva
Čtení (r) Uživatelé mohou pouze číst text. Pokud je uživatel v privátních uživatelích, uvidí také privátní text.
Zápis (w) Uživatelé mohou upravovat a číst text. Mohou také definovat privátní text a uživatele resp. skupiny, které tento text uvidí.
Kontrola (c) Obdobně jako právo zápisu.
Modul je nezávyslý.

Úprava

Text na této stránce můžete upravit pomocí nástrojů a položky Upravit text.

Metadata
